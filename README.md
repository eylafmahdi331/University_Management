This project demonstrates the creation and management of a University_Management 
system using the Kali Linux Command Line Interface (CLI). The directory structure 
simulates a university environment, including Courses, Attendance, Reports, and Tools, 
with multiple nested folders and files representing assignments, grades, syllabi, and 
automation scripts. 
The project highlights the use of absolute and relative paths, file and directory 
manipulation, permission management, and automated task scheduling, showcasing 
practical CLI skills in organizing and maintaining a complex file system efficiently.
